# End-To-End-Waste-Detection-Using-YOLO

# Waste Detection using YOLOv11

## Features
- Waste classification
- Image upload
- Real-time detection
- Streamlit web interface

## Dataset
- 11,466 training images
- 1,092 validation images
- 42 classes

## Model
- YOLOv11n
- Trained for 50 epochs
- NVIDIA RTX 3050

## Results
- mAP@50 = 40.1%
- mAP@50-95 = 29.4%
